## ðŸš€ Getting Started

Follow these steps to download and run the application:

1. **Visit the Release Page**  
   Go to the releases page to access the downloadable files. You can find it [here](https://github.com/Saurav0989/node-fastify-starter/releases).

2. **Choose Your Version**  
   On the releases page, you will see various versions of the kit. Locate the latest version and click on it to view the details.

3. **Download the Kit**  
   Scroll down to the "Assets" section of the release details. Look for a file that ends with `.tar.gz` or `.zip`. Click on it to download the file to your computer.

4. **Extract the Files**  
   Once the download is complete, find the file in your computer's downloads folder. Depending on your operating system:
   - **Windows:** Right-click the file and select "Extract All", then follow the prompts.
   - **Mac:** Double-click the downloaded file. It will automatically extract to a folder.
   - **Linux:** Open a terminal and use the command `tar -xvf yourfile.tar.gz` to extract.

5. **Install Dependencies**  
   After extracting, navigate to the extracted folder. Open a terminal or command prompt and execute the following command to install the necessary dependencies:
   ```
   npm install
   ```

6. **Run the Application**  
   With everything set up, you can now run your application. Use the command:
   ```
   npm start
   ```
   This command will start the server, and you will see a message indicating that the server is running.
